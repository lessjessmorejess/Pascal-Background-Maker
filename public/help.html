Help & FAQ


Why is the text going outside of the bubble?

All the quotes end up wrapping to new lines at different points, so by default they may not look quite right. Try toggling the Line Wrap or Font Size to see if that solves the issue

My screen size isn't an option?

Unfortunately, it's impossible to add every screen size possible. Try finding the closest resolution to your phone, or if you're used ot GitHub, report it here!

I picked my iPhone model, but the image is too big?

Try zooming the image out and it should fit properly. By default iOS sets the Live Zoom to be on, so turn that off and zoom out and it shoudl be correct!

Why is there so much space around the letters in the bubble. How can I fix it?

This can be caused by your web browser. This app was built around Google Chrome and Safari on iOS.

How can I make my background cycle between quotes/themes automatically on iOS?

You'll need to use the ShortCuts app and create your own automation. This guide should help with that!
